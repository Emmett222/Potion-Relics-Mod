# Potion-Relics-Mod

## What does this mod do?
Potion Relics Mod adds relics to the game that give you infinite of a potion effect as long as it is in your inventory.

## Offhand upgrading
Relics can be upgraded one level in the offhand.

## Config
These properties of the relics can be changed in the config file.
Each individual relic has these settings:
- Potion level: Changes the level of the effect.
- Offhand upgrading: Changes if the relic can be upgraded in the offhand or not.
- Particles: Change if the relic shows particles or not.

An exemption to these is the Absorption Relic.
It has these settings:
- Absorption cooldown: How long a player has to wait before Absorption is given to them again.
- Absorption amount: How many absorption hearts the player with the relic gets.
